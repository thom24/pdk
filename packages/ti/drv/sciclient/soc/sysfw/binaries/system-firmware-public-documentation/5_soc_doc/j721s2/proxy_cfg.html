J721S2 Proxy Device Descriptions¶

Introduction¶

This chapter provides information on the Proxy devices in the J721S2 SoC. Some System Firmware TISCI messages take device specific inputs. This chapter provides information on the valid values for Proxy TISCI message parameters.

Proxy Device IDs¶

Some System Firmware TISCI message APIs require the Proxy device ID be provided as part of the request. Based on J721S2 Device IDs these are the valid Proxy device IDs.

Proxy Device Name Proxy Device ID
J721S2_DEV_NAVSS0_PROXY_0 256
J721S2_DEV_MCU_NAVSS0_PROXY0 271

Proxy Indices¶

This section describes Proxy indices. The proxy index is used in some Proxy based TISCI messages.

Warning

Resources marked as reserved for use by TIFS cannot be assigned to a host within the RM Board Configuration resource assignment array. The RM Board Configuration is rejected if an overlap with a reserved resource is detected.

Proxy Name Proxy Range
J721S2_DEV_NAVSS0_PROXY_0 0 to 63
J721S2_DEV_MCU_NAVSS0_PROXY0 (RESERVED BY SYSTEM FIRMWARE) 0 to 0
J721S2_DEV_MCU_NAVSS0_PROXY0 1 to 63
