AM62X Secure Proxy Descriptions¶

Introduction¶

This chapter provides information of Secure Proxies and communication paths that are permitted in the AM62X SoC. These host IDs represent processing entities (or PEs) which is mandatory identification of a Host in a processor. See PE/Host documentation for further information

Enumeration of Secure Proxies¶

Sproxy ID Sproxy Name
0 DMASS0_SEC_PROXY_0
1 SA3_SS0_SEC_PROXY_0
