AM62X Board Configuration Resource Assignment Type Descriptions¶

Introduction¶

This chapter provides information of Board Configuration resource assignment type IDs that are permitted in the AM62X SoC. The resource type IDs represent AM62X resources ranges assignable to SoC processing entities (or PEs).

WARNING: System Firmware RM currently supports a maximum of 305 RM board configuration resource assignment ranges on the AM62X SoC. Sending more entries than the maximum will result in the RM board configuration being NACK’d
